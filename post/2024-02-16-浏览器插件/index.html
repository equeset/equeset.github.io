浏览器常用插件

by Septem
2024-02-16
computer index

1. 视频浏览

油猴（篡改猴）+HTML 5 播放助手:可以调节速度、亮度等基本视频信息，注意快捷键或许与 Vimium 冲突。适用所有不想点来点去调节视频速度且甚至想要调到 2 倍速以上的场景，非常好用。

Open-as-Popup：把当前窗口单独弄出去，更简洁且仍可使用网页插件其他插件。适用于屏幕不那么大想尽可能榨干空间的情况（？）

FireShot：截图插件，免费够使。

2. 阅读与信息

Tabcopy：将网址复制为 md 格式，会配合 Obsidian 进行使用。

简悦：很牛的阅读插件，但是和我相性不好……作者的教程设计非常执迷于连续跳转（即跳到 A 网页以后跳到了 B 又显示还有别的相关内容在 C，于是你就要从 C 做起但是门外汉搞 C 可能又要经历一遍。），经常给我跳蒙……黑名单经常失灵，也不知道是我写得不对还是怎样。保存到 Obsidian 的功能倒是比较好使，基本只是在用这个功能。心情复杂……

Roam-Highlighter：如果你习惯网页直接阅读并且复制下关键内容可以使用这个，萤光笔一口气划够，之后全部复制走。

Vimium：浏览器快捷键。使用键盘操作浏览器的插件。主要被我用于给长长的网页加入书签，下次跳转继续阅读。如果快捷键冲突，可以在不想使用它的网页点击它的图标然后 add rules。 ^988abc

3. 浏览器管理

Toby：管理网页，一种更方便的书签。基本可以按照类别存放，很便利，是实际使用频率最高的。

SimpleExtManager：管理浏览器插件的插件（套娃）。更好地打开、关闭插件，可以给插件设置群组，集体打开和关闭，好使。

4. 翻译

沉浸式翻译：每段挨个翻译，可以对比着看。缺点是会影响下面这个插件的使用……得选显示原文，否则无法正常加载字幕。

Language Reactor：油管中英对比字幕，可以点击查看单词含义&发音。更多功能如导出到 anki 需要付费。此处也推荐一个可以下载油管等视频网站下载字幕的网站 https://downsub.com/ 。

5. 下载

IDM：什么？还有人没用 IDM 吗？IDM 可以联动油猴搭配 API 下载插件去下某度大文件。嗯……省得开客户端（话说某度的客户端根本就是在偷摸挖矿吧）。

Billfish：搭配本地 app 一键把图拉下存储到本地。是免费版的那啥，那啥来着？忘了。
